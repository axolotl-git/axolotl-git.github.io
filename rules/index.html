Yantra Launcher Script Hub

An unofficial project aimed at gathering all scripts made for Yantra Launcher by the community in one place, easily accessible to everyone.

Platform Rules

  1. Security: Scripts must be free of vulnerabilities and must not contain malicious or potentially malicious code.
  2. Nomenclature: Use clear and consistent naming conventions for variables, functions, and modules to improve the readability of your code.
  3. Testing: Before uploading a script, it should be tested in a development environment to ensure that it works as expected and does not generate errors.
  4. License: Scripts must be open source and respect copyright.
  5. Feedback: Script authors should be open to feedback and ready to fix any issues reported by users.

These rules help maintain order and ensure the platform's quality and security.
