## Introduction

Welcome to the **Yantra Launcher Script Hub**, an unofficial project that gathers all scripts created for Yantra Launcher by the community, making them easily accessible to everyone. This README file provides instructions on how to modify the project to add new scripts and manage their downloads.

## Add your scripts

if you want to add scripts just create a fork of this and modify `scripts.json` then you can make a pull request 

To do that you need to follow this structure
```
{
  "Script name": "https://example.com/path/to/script1.lua"
}
```

`Script Name` is obliviusly the name of the script and `https://example.com/path/to/script1.lua` is the link to download
